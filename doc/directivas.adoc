= Directivas reconocidas

La siguiente tabla lista las directivas que modifican el comportamiento del ensamblador Lily.

|===
| Categoría                | Expresión     | Significado                | Descripción

.11+| Definición de constantes
| `DB _exp_`
| Define Byte
| Define un entero de 1 byte

| `DWL _exp_`
| Define Word (Least)
| Define una palabra (2 bytes), empezando por el byte menos significativo

| `DWM _exp_`
| Define Word (Most)
| Define una palabra (2 bytes), empezando por el byte más significativo

| `DDL _exp_`
| Define Double word (Least)
| Define una palabra doble (4 bytes), empezando por el byte menos significativo

| `DDM _exp_`
| Define Double word (Most)
| Define una palabra doble (4 bytes), empezando por el byte más significativo

| `DQL _exp_`
| Define Quad word (Least)
| Define una palabra cuádruple (8 bytes), empezando por el byte menos significativo

| `DQM _exp_`
| Define Quad word (Most)
| Define una palabra cuádruple (8 bytes), empezando por el byte más significativo

| `DR _exp_`
| Define Real
| Define un valor de punto flotante de precisión simple, según el estándar IEEE 754

| `DRD _exp_`
| Define Real Double
| Define un valor de punto flotante de precisión doble, según el estándar IEEE 754

| `DRL _exp_`
| Define Real Long
| Define un valor de punto flotante de 80 bits, según la implementación en el Intel 8087

| `DFS _exp_`
| DeFine Storage
| Define una secuencia de bytes nulos

.2+| Definición de símbolos
| `CONST _exp_`
|
| Define una constante conocida por el preprocesador

| `VAR _exp_`
|
| Define una variable conocida por el preprocesador

.7+| Condicionales
| `IF _exp_`
|
| Bloque para iniciar un condicional

| `IFDEF _eti_`
| IF DEFined
| Bloque para iniciar un condicional que comprueba si una etiqueta está definida

| `IFNDEF _eti_`
| IF Not DEFined
| Bloque para iniciar un condicional que comprueba si una etiqueta no está definida

| `ELSE`
|
| Bloque para la condición contraria de un condicional

| `ELIF _exp_`
| ELse IF
| Bloque para una condición adicional

| `ELIFDEF _exp_`
| ELse IF DEFined
| Bloque para una condición adicional que comprueba si una etiqueta está definida

| `ELIFNDEF _exp_`
| ELse IF Not DEFined
| Bloque para una condición adicional que comprueba si una etiqueta no está definida

//.2+| Bucles
//| WHILE exp
//|
//| Bloque para un bucle
//
//| LOOP
//|
//| Marca de fin de un bloque WHILE
//
| Inclusión de archivos
| `INC _exp_`
| INClude
| Indicador para incluir externamente un archivo

.2+| Control de ejecución
| `ORG _exp_`
| ORiGin
| Indicador para el lugar de inicio de ejecución

| `STOP`
| Fin de programa
| Indicador para dejar de procesar un programa

.2+| Estructuras de datos
| `STRUCT`
|
| Bloque para una estructura de datos

| `UNION`
|
| Bloque para una unión

.2+| Subrutinas
| `MACRO`
|
| Bloque para una macro

| `PROC`
| PROCedure
| Bloque para una subrutina

| Miscelánea
| `END`
|
| Marca de fin para un bloque de directiva `IF`, `IFDEF`, `IFNDEF`, `ELSE`, `ELIF`, `STRUCT`, `UNION`, `MACRO` y `PROC`

|===
